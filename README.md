# IT & Cybersecurity Attorney - Legal Consulting Agency

A comprehensive legal services platform specializing in IT law, cybersecurity compliance, and technology legal consulting.

## Features

- **Legal Services**: IT law, cybersecurity compliance, technology contracts
- **Business Directory**: Searchable database of registered businesses
- **Financial Services**: Trust management, power of attorney services
- **Resources**: Legal documentation and compliance guides
- **Analytics Dashboard**: Business intelligence and compliance tracking
- **Enterprise Solutions**: API integration and custom solutions

## Tech Stack

- **Framework**: Next.js 15.5.4
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL
- **Runtime**: Node.js 20

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
Create a `.env` file with your database connection:
```
DATABASE_URL=your_postgresql_connection_string
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:5000](http://localhost:5000) in your browser

## Database Setup

To seed the database with initial data:
```bash
npm run db:push
```

## Project Structure

- `/pages` - Next.js pages and API routes
- `/components` - Reusable React components
- `/styles` - Global styles and CSS
- `/lib` - Database and utility functions
- `/scripts` - Database seeding scripts
